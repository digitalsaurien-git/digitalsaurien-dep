# 🦎 ReptilTrack

**ReptilTrack** est une application web de gestion de collection de reptiles moderne, mettant l'accent sur l'esthétique, le suivi de santé précis et l'optimisation énergétique. Conçue pour les éleveurs passionnés et les conservateurs de spécimens exotiques.

![Aperçu de ReptilTrack](https://placeholder-url.com/dashboard-preview.png)

## ✨ Fonctionnalités Clés

*   **Dashboard "Biophilic Curator"** : Une vue d'ensemble premium de votre élevage avec statistiques de santé et de coûts en temps réel.
*   **Gestion de Collection (Specimen Tracking)** : Fiche détaillée pour chaque animal, incluant le nom scientifique, le numéro de puce/CITES et le statut de santé.
*   **Journal de Soins Historique** : Suivez les repas, les mues, les pesées et les traitements médicaux pour chaque spécimen.
*   **Monitorage de l'Habitat** : Gérez vos terrariums, la population par unité et les spécifications climatiques.
*   **Calculateur d'Énergie & Coûts** : Estimez le coût quotidien et mensuel de votre matériel électrique (lampes, tapis chauffants, brumisateurs) grâce à un calculateur intégré.
*   **Interface Responsive Premium** : Une expérience "glassmorphism" fluide, optimisée aussi bien pour le bureau que pour la saisie rapide sur mobile.

## 🎨 Design Système : "The Biophilic Curator"

L'interface utilise un thème **Navy-Emerald** sophistiqué, inspiré par les expositions de musées d'histoire naturelle :
- **Palette** : Base Navy-Charcoal (#0f141a), accents Émeraude (#4edea3) et Bleu Ciel (#adc6ff).
- **Glassmorphism** : Effets de flous arrières et bordures semi-transparentes pour une sensation de profondeur "aquarium".
- **Typographie** : *Plus Jakarta Sans* pour l'autorité des titres, *Manrope* pour la précision technique des données.

## 🚀 Installation & Lancement

L'application est construite avec **Vite**, **React** et **Lucide-React**.

### Pré-requis
- Node.js (v18+)
- npm / yarn

### Installation
```bash
git clone https://github.com/votre-compte/reptil-track.git
cd reptil-track
npm install
```

### Développement
```bash
npm run dev
```

### Construction (Production)
```bash
npm run build
```

## 🛠️ Stack Technique

- **Framework** : React 18+
- **Styling** : Vanilla CSS 3 with Modern Design System (Glassmorphism)
- **Icônes** : Lucide React
- **Routage** : React Router DOM
- **État** : React Context API
